# Voxfeed

Live Demo: https://voxfeed.vercel.app

Voxfeed is a real-time social media platform designed for seamless interaction. It allows users to share thoughts, follow others, and engage with content through likes and comments in a fast, responsive environment.

---

## Features

### User Interactions

- Create, edit, and delete posts with image support
- Real-time like and comment system
- Bookmark posts for later viewing
- Follow/Unfollow system for personalized feeds

### Real-time & Discovery

- Instant notifications for likes and follows
- Global search for finding users and posts
- Real-time activity feed via Pusher
- Trending/Suggested users discovery

---

## Tech Stack

**Frontend:** Next.js (App Router), Tailwind CSS  
**Backend:** Next.js Server Actions, Route Handlers  
**Database:** PostgreSQL (via Prisma ORM)  
**Real-time:** Pusher  
**Auth:** Better-Auth  
**Deployment:** Vercel
